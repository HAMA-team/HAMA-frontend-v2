## 🎯 에이전트 응답 전략 제안

### 1. 컨텍스트 인식

- 초보자 감지 → 쉬운 설명, 단계별 가이드
- 손실 상황 감지 → 심리적 지원 + 객관적 조언
- 긴급 상황 감지 → 빠른 의사결정 지원

### 2. 개인화

- 투자 성향 파악 (공격적/안정적)
- 투자 기간 고려 (단기/장기)
- 자금 규모 반영

### 3. 리스크 관리

- 항상 리스크 경고 포함
- 분산투자 권유
- 손실 한도 설정 안내

### 4. 정보 제공 수준

- 초보자: 핵심만 간단히 + 추가 학습 자료
- 중급자: 상세 분석 + 여러 관점
- 고급자: 심화 데이터 + 전문 용어 사용
